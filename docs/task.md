# Cross Chat — 当前任务 (Task Board)

> 更新: 2026-06-03 | 状态: 方案已确定，逐步增量开发

---

## 产品目标

直播时，**主播说的话** 和 **观众弹幕** 进行智能匹配，匹配到的弹幕在 OBS 叠加层中高亮显示，让观众一眼看出"主播在回复谁"。

## 数据流

```
B站直播间 ──弹幕──→ bililive_dm ──桥接──→ Python 后端 ──WS──→ 前端 OBS 叠加层
                                                      ↑
麦克风 ──语音──→ obs-localvocal ──字幕文本─────────────┘
                                              (匹配引擎)
```
